PROJECT TITLE: Sports League
PURPOSE OF PROJECT: To implement and test the working of different Java functions using the English Football League as an example
VERSION or DATE: 20th October 2022
HOW TO START THIS PROJECT: Blue J or Eclipse
AUTHORS: Saivenket Patro K

USER INSTRUCTIONS: 
For Blue J, double click on the package file in the root directory. 
And right click on the League_Manager class, click on the main option.
For Eclipse, open the League_Manager class using the IDE and click on run option on the menu bar.
